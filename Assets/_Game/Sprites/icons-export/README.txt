FIGMA
  Drag any .svg into Figma — imports as editable vector layers (recolor strokes,
  adjust the glow blur/shadow, convert to components). A native .fig cannot be
  generated here; SVG import is the supported path.

UNITY
  Use the PNGs. Texture Type: Sprite (2D and UI), PPU 100, Filter: Point,
  Compression: None, Mip Maps: Off, Wrap: Clamp.
  9-SLICE (border 24): Panels, Buttons, ProgressBars, Frames, HUD_Frame, Toggle BG.
  SIMPLE: icons, hearts, stars, badges, dots, glows, decorations, sliders.
